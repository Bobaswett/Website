4 Wheel Autonomous Rover Hand Gesture Tracking and Control
  • GitHub

    • This platform was built for an unsupervised algorithms class, but I have also used it for various other projects. In this project I utilized PCA and Clustering Algorithms to train the robot to track hand gestures. I utilized a Jetson Nano, ESP32, Motors, Servos, and I2C to bring this project to life. See details below!

    The body of the rover was 3D printed using a Sovol printer. I designed the pieces so they could hold my electronics, but I elected to keep everything in the bread board so I can reuse parts. The motors are controlled using PWM from the ESP32 hooked up to the motor drivers. The Jetson Nano connects to the ESP32 with UART and this allows for the Jetson to perform the Machine Learning algorithm then send commands to the ESP32. The Pan and Tilt servos are controlled with PWM as well, but are instead hooked up to a board with an I2C connection from the Jetson Nano. The camera is set up to track hands, then the Machine Learning Algorithm categorizes the command as left right or back. The algorithm uses principal component analysis to reduce the number of features in each frame, then groups each frame into trained clusters from over 300 training images. All code and data is available at the Github link above.
